# Book Recommendation System

A personalized book recommendation system built using Item-Based Collaborative Filtering, generating top-5 book suggestions based on user rating patterns.

## What it does

- Cleans and processes the 1.5M-row Book-Crossing dataset using Pandas and NumPy
- Builds a user-item ratings matrix
- Uses cosine similarity to find and rank similar books
- Generates personalized top-5 book recommendations for a given user

## Tech stack used

- Python
- Pandas, NumPy
- Scikit-learn (cosine similarity)
- Flask (web interface)
## 🏃 How to Run
1. Clone the repo: `git clone https://github.com/PalakA321/Book_Recommendation_System`
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python app.py`
